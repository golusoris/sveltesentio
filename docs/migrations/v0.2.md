## Codemod runner

When at least one row uses `Codemod: auto`:

```bash
pnpm dlx @sveltesentio/migrate@0.2 run --from 0.1 --to 0.2
```

The `@sveltesentio/migrate` package wraps `jscodeshift` + `ast-grep` rules under `tools/migrations/v0.2/`. If no `auto` rows exist, the package is not published for this release.

## Verifying the migration

After applying every row above:

1. `pnpm typecheck` — no new errors introduced.
2. `pnpm test` — full suite green.
3. `pnpm exec playwright test` — e2e suite green.
4. `pnpm audit` — no new vulns introduced by transitive bumps.
5. Re-run the [downstream-antipatterns checklist](downstream-antipatterns-v0.1.md) — every entry must still pass (the v0.1 checklist remains binding through v0.2 unless explicitly superseded by a row above).
