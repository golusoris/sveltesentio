# Code of Conduct

This project adopts version 2.1 of the Contributor Covenant, verbatim.

Canonical text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
Plain-text copy: <https://www.contributor-covenant.org/version/2/1/code_of_conduct.txt>

## Applicability

The Covenant applies to:

- every public space managed by this project (GitHub issues, pull requests,
  discussions, code comments, releases, and any future community channel);
- every private channel used to represent the project (email, chat, DMs
  tied to maintainer duties);
- every public event where a participant represents sveltesentio.

## Reporting

Report Covenant concerns to the maintainer at **lusoris@pm.me**.

Reports are acknowledged within 48 hours. Reporter confidentiality is
maintained.

For security-vulnerability reports see [SECURITY.md](SECURITY.md) instead —
the triage channel is different.

## Enforcement ladder

Enforcement follows the Covenant's **Community Impact Guidelines** (section
"Enforcement Guidelines" of the canonical text): correction → warning →
temporary ban → permanent ban. The maintainer is the sole enforcement
authority pending the project graduating to an organization with a wider
moderation team.
